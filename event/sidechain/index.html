A Sidechain implementation without Bitcoin protocol modification

Friday, August 28nd 2014 at 8 PM EDT

Online event only at this page

Phil Champagne will be presenting live the talk he has given at Cryptolina Expo 2014. This open invitation will give anyone interested the opportunity to watch the presentation and participate in the Q&A session at the end.

The presentation will discuss a theoretical strategy to implement a sidechain utilisation without any modification of the Bitcoin software protocol. This type of sidechain could have major implications for the expansion of the blockchain beyond the recording of currency transactions. It will open up a vast array of application for new business opportunities and innovative tools and services to explore.

From a personal email exchange between Phil Champagne and Adam Draper:

"I think it is almost time for this type of tech, so if you have the ability to build it, definitely go for it."

- Adam Draper  

So, we would like to invite all developers interested to participate in the implementation of this project and entrepreneurs interested by all the consequent social implications to watch this presentation.

Hope to see you there!
